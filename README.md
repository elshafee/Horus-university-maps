## 📖 Overview

The **Horus University Campus 3D Map** is a cutting-edge web application designed to provide students, faculty, and visitors with an immersive, interactive, and visually stunning way to navigate the entire campus. This innovative platform reimagines campus exploration by combining modern web technologies with intuitive user experience design.

Built with a premium UI/UX design language featuring the deep navy and "Horus Gold" aesthetic, this map transcends traditional navigation by offering a full 3D environment, detailed building insights, and an engaging interface that makes discovering campus facilities a delightful experience.

## ✨ Key Features

- 🗺️ **Interactive 3D Navigation**: Pan, tilt, and zoom through a fully realized 3D representation of the Horus University campus.
- 🏢 **Building-Specific Details**: Click on individual campus buildings to reveal detailed information, including faculty names, facilities, and contact details.
- 🔍 **Smart Search**: Quickly find specific faculties, halls, or points of interest using the built-in, lightning-fast search engine.
- 🚁 **Drone Aerial View**: Experience the campus from a breathtaking aerial perspective with the exclusive virtual flight mode.
- 🎨 **Premium Aesthetics**: A custom "Deep Navy & Horus Gold" theme with micro-animations and glassmorphism elements, delivering a luxury user experience.
- 📱 **Fully Responsive**: Optimized for seamless performance on desktops, tablets, and mobile devices.

## 🛠️ Technologies Used

This project is built using a modern, lightweight, and performant tech stack:

- **HTML5 & CSS3**: For structure, styling, and fluid animations.
- **Vanilla JavaScript (ES6+)**: Providing the core logic without the overhead of heavy frameworks.
- **MapLibre GL JS**: Powering the high-performance 3D map rendering and interactive geospatial features.
- **Custom Design System**: Utilizing bespoke CSS variables for consistent theming and micro-interactions.

## 🚀 Getting Started

### Prerequisites

Since this is a static web application, you only need a modern web browser (Chrome, Firefox, Safari, or Edge) to run it locally.

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/elshafee/Horus-university-maps.git
   ```

2. **Navigate to the project directory:**
   ```bash
   cd Horus-university-maps
   ```

3. **Run the application:**
   - **Option 1 (Simple):** Double-click on `index.html` or `campus-map.html` to open it in your default browser.
   - **Option 2 (Live Server):** If you are using VS Code, install the "Live Server" extension, right-click `index.html`, and select "Open with Live Server". This ensures all local assets load correctly and provides live reload functionality.
   - **Option 3 (Python Server):** Open a terminal in the project directory and run:
     ```bash
     python -m http.server 8000
     ```
     Then open `http://localhost:8000` in your browser.

## 🎮 Usage

### Navigation Flow
- `index_launcher.html` / `index.html` - The main entry point and landing experience.
- `campus-map.html` - The core interactive 3D map interface.

### Map Controls
- **Left Click & Drag:** Pan across the map.
- **Right Click & Drag (or CTRL + Left Click):** Adjust pitch (tilt) and bearing (rotation) for 3D views.
- **Scroll Wheel:** Zoom in and out.
- **Search Bar:** Use the sidebar or search feature to locate buildings, faculties, or points of interest.
- **Building Click:** Click directly on 3D models to access detailed information panels.
