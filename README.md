## 📖 Overview

The **Horus University Campus 3D Map** is a cutting-edge web application designed to provide students, faculty, and visitors with an immersive, interactive, and visually stunning way to navigate the university campus.

Built with a premium UI/UX design language featuring the deep navy and "Horus Gold" aesthetic, this map transcends traditional navigation by offering a full 3D environment, detailed building insights, and advanced search capabilities.

## ✨ Key Features

- 🗺️ **Interactive 3D Navigation**: Pan, tilt, and zoom through a fully realized 3D representation of the Horus University campus.
- 🏢 **Building-Specific Details**: Click on individual campus buildings to reveal detailed information, including faculty names, facilities, and contact details.
- 🔍 **Smart Search**: Quickly find specific faculties, halls, or points of interest using the built-in, lightning-fast search engine.
- 🚁 **ESSIC Drone Feature**: Experience the campus from a breathtaking aerial perspective with the exclusive "ESSIC drone" virtual flight mode.
- 🎨 **Premium Aesthetics**: A custom "Deep Navy & Horus Gold" theme with micro-animations and glassmorphism elements, delivering a luxury user experience.
- 📱 **Fully Responsive**: Optimized for seamless performance on desktops, tablets, and mobile devices.

## 🛠️ Technologies Used

This project is built using a modern, lightweight, and performant tech stack:

- **HTML5 & CSS3**: For structure, styling, and fluid animations.
- **Vanilla JavaScript (ES6+)**: Providing the core logic without the overhead of heavy frameworks.
- **MapLibre GL JS**: Powering the high-performance 3D map rendering and interactive geospatial features.
- **Custom Design System**: Utilizing bespoke CSS variables for consistent theming and micro-interactions.

## 🚀 Getting Started

### Prerequisites
Since this is a static web application, you only need a modern web browser (Chrome, Firefox, Safari, Edge) to run it locally.

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/your-username/Horus-university-maps.git
   ```
2. **Navigate to the project directory:**
   ```bash
   cd Horus-university-maps
   ```
3. **Run the application:**
   - **Option 1 (Simple):** Just double-click on `index.html` or `campus-map.html` to open it in your default browser.
   - **Option 2 (Live Server):** If you are using VS Code, install the "Live Server" extension, right-click `index.html`, and select "Open with Live Server". This ensures all local assets load correctly.

## 🎮 Usage

- **Navigation Flow:**
  - `index_launcher.html` / `index.html` - The main entry point and landing experience.
  - `campus-map.html` - The core interactive 3D map interface.
- **Controls:**
  - **Left Click & Drag:** Pan across the map.
  - **Right Click & Drag (or CTRL + Left Click):** Adjust pitch (tilt) and bearing (rotation) for 3D views.
  - **Scroll Wheel:** Zoom in and out.
- **Interactions:** Use the sidebar or search bar to locate buildings, or click directly on the 3D models to access the information panels.
